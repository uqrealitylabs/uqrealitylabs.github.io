## Quick Setup

```sh
npm install
npm run dev
```

## Main Commands

```sh
npm run validate:content
npm run check:no-js
npm run test:unit
npm run build
npm run ci
```

## Repo Notes

The repo is for code, config, scripts, generated schemas, and small entry docs.
Human-facing setup, architecture, content editing, CI, security, and benchmark notes belong in the wiki.

Security reports should not go through public issues. Use the repo security policy.
