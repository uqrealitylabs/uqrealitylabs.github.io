# eyslie

Clean-room TypeScript xeyes-style helpers for constrained pupils, hide radius,
wink timing, and simple expression states.

The package does not copy xeyes source. The xorg xeyes project was reviewed as
prior art and is identified upstream as X11 licensed.

```ts
import { constrainPupilOffset, getOrganicWinkDelayMs } from "eyslie";
```
