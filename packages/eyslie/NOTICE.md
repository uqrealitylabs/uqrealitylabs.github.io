eyslie is a clean-room TypeScript implementation inspired by the xeyes
"follow the mouse" behaviour. It does not copy source code or algorithm text
from xeyes.

The upstream xorg xeyes project was reviewed as prior art in July 2026. It is
published by the X.Org project and identified on GitLab as X11 licensed:
https://gitlab.freedesktop.org/xorg/app/xeyes
