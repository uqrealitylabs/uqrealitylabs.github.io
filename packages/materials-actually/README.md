# materials-actually

Tiny TypeScript helpers for pointer-local material interaction: poke pressure,
velocity, local influence, material behaviour metadata, and optional haptic
patterns.

```ts
import {
  applyPoke,
  createPokeState,
  materialHaptics,
} from "@uqrealitylabs/materials-actually";
```

Build with `npm run build -w @uqrealitylabs/materials-actually`; the package exports `dist`.
