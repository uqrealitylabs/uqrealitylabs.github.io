Code map:
- src/app/: React shell.
- src/content/: JSON content graph, schema, registry, and renderer.
- src/features/: lazy visual scene and extracted interaction helpers.
- src/shared/: i18n, coarse state, Tailwind entry, tokens, and base styles.
- scripts/: TypeScript-only repo checks, asset work, content validation, and benchmarks.
- tests/: Vitest unit tests plus Cypress component/E2E specs.
- public/: static assets and generated public files only.
- .github/workflows/: CI, security, benchmarks, and Pages deploy.
